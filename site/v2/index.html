Morris-Lecar model at Hopf bifurcation

This shows a single Morris-Lecar model undergo a Hopf bifurcation as input voltage is slowly ramped up.

Keys: Click and hold anywhere in the canvas area to let input voltage ramp up. Release to reset voltage to rest.

You can see the input voltage change reflected in the red meter in the top left corner.

As input voltage ramps up, there is a threshold past which the model will start oscillating out of nowhere. Oscillations start at low amplitude and slowly increase in amplitude as input voltage increases, as is characteristic of a Hopf bifurcation in the phase space.

Once the mouse button is released and input voltage is instantaneously reset to 0, the periodic orbit (oscillation) resulting from the Hopf bifurcation immediately vanishes and oscillations become increasingly damped as the model approaches its stable equilibrium.
